# PzGame

Гра, де треба керувати синім колом і збирати монетки. Червоне коло — ворог, воно женеться за гравцем зі швидкістю 120 px/сек. Якщо ворог торкнеться — гра закінчується. За кожні 5 очок ворог рухається швидше на 10 px/сек. Коли береш монету, чути короткий звук. Далі можна додати меню, кілька рівнів або більше ворогів.
